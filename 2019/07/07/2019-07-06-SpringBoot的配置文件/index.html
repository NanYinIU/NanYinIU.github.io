SpringBoot中使用配置

一、配置文件

SpringBoot使用全局的配置文件，配置文件名是固定的

  • application.properties
  • application.yml

使用配置文件可以更改系统自动配置的默认值，比如端口号等信息。

YAML

yaml以数据为中心，比xml，json更适合做配置文件。
